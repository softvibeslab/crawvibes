- **Browser Automation**: Built-in Puppeteer support
- **Web Gateway**: Access via web interface

## Quick Start on EasyPanel

1. In EasyPanel, create a new service
2. Select this template or paste the Git URL: `https://github.com/softvibeslab/crawvibes.git`
3. Navigate to the `openclaw` folder
4. Fill in the configuration form
5. Deploy!

## Configuration

### Required

- **Gateway Token**: Generate a secure random token
- **API Key**: Based on your selected model (Anthropic, OpenAI, Zhipu, Google, or xAI)

### Optional

- **Telegram Bot Token**: Get from [@BotFather](https://t.me/botfather)
- **Timezone**: Your preferred timezone

## Access

After deployment, access OpenClaw at: `https://your-domain.com`
